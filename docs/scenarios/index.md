# Типовые сценарии

Каждый сценарий должен описывать реальную задачу пользователя, а не только отдельную кнопку или экран.

## Рекомендуемая структура сценария

1. Когда используется сценарий
2. Что нужно до начала
3. Пошаговое выполнение
4. Что получается в результате
5. Какие функции связаны со сценарием
