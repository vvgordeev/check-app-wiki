# Product Wiki

Это пользовательская база знаний по продукту. Она предназначена для конечных пользователей, администраторов и команды внедрения.

## Что здесь есть

- описание ключевых функций
- пошаговые сценарии использования
- пояснения по ролям и доступам
- иллюстрации экранами приложения
- сквозной поиск по всей базе

## Как поддерживается актуальность

Wiki обновляется из репозитория автоматически:

1. изменения в коде сопоставляются с функциями продукта
2. для затронутых функций формируются изменения в статьях
3. переснимаются нужные экраны
4. в документации создаётся pull request на проверку
