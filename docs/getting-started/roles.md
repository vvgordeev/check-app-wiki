# Роли пользователей

В этом разделе описываются роли и их права.

## Рекомендуемый шаблон

| Роль | Что может делать | Где используется |
| --- | --- | --- |
| Пользователь | Выполняет повседневные операции | Основные сценарии |
| Руководитель | Контролирует и анализирует | Отчёты, мониторинг |
| Администратор | Настраивает систему и доступы | Администрирование |
